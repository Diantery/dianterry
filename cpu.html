Ideas From CPU

Yups CPU, maybe you have potato inside

Pengertian CPU :

CPU atau Central Processing Unit adalah bagian terpenting dalam sistem komputer yang dimana CPU akan melakukan dan menjalankan segala instruksi yang ada. CPU bisa di bilang sebagai otak dari komputer itu sendiri karena tugas nya yaitu komputasi data yang di perintahkan oleh pengguna.

Untuk CPU sendiri terbagi menjadi banyak jenis. Ada CPU generasi pertama, generasi kedua, dan lain sebagai nya yang dimana sempat kami jelaskan pada link ini. Dari beberapa generasi tersebut, CPU melahirkan beberapa generasi terbaru seperti CPU 64 Bit, CPU ARM atau RISC-V, dan jenis CPU lain nya.

Komponen Utama :

Komponen utama CPU ada pada Control Unit atau I/O yang bertugas , ada Arithmetic Logic Unit atau ALU yang dimana bertugas untuk mengolah data, ada Registers yang berfungsi sebagai media penyimpanan internal di dalam CPU itu sendiri, ada CPU Interconnection atau BUS Data yang fungsi nya adalah menghubungkan komunikasi data kepada Control Unit, ALU, dan register yang dimana BUS data tersebut merupakan jalan besar bagi data yang akan melalui nya.

CPU dibagi menjadi dua kategori, ada Single Core CPU dan Multi Core CPU. Perbedaan yang signifikan dari kedua nya adalah jumlah Compute Unit yang ada di dalam Satu Package CPU itu sendiri. Single Core CPU hanya berisi satu buah Compute Unit atau CPU itu sendiri di dalam Satu Package nya. Berbeda dengan Multi Core CPU yang dimana di dalam Satu Package itu sendiri bisa berisikan 2 Compute Unit, 4 Compute Unit, atau lebih dengan syarat 2^n atau dalam artian tidak ganjil.

Single Core CPU :

Seperti yang sudah di jelaskan sebelum nya, di dalam Satu Package Single Core CPU hanya berisikan Satu Unit Compute Modul atau CPU itu sendiri, yang dimana CPU itu terdiri dari Control Unit, ALU, Register, Bus Data.

Multi Core CPU :

Dan untuk Multi Core CPU, inti dari isi di dalam Compute Modul nya serupa. Terdiri dari Control Unit, ALU, Register, dan Bus Data. Namun yang membedakan nya adalah jumlah Control Unit, ALU, dan Register lebih banyak dan tergantung berapa Core CPU tersebut. Contoh soal CPU itu 4 Core, maka Compute Modul yang berisikan Control Unit, ALU, Register tersebut ada 4.
